myGitFolio

Some of the greatest development work is done in open source, and you should get credit for it.
But when you apply for a job, and you want to list some projects you've worked on, it's suboptimal to just
give the employer your github link. Thus, a solution was born : myGitFolio.
myGitFolio is a incredibly customizable resource that will gather
all of your GitHub repositories, order them in a way you want and 
place them all in a file, with a concise description of the project.

and the best part...

myGitFolio is open source! You can see how it's done, and even fork off your own version.
If you spot a flaw, you can fix it and merge back! This way there are no worries about what we do with your data,
you already know! So what are you waiting for? Let's see what you do!
